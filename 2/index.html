NES homebrew experiment #2: compile JS code into a NES rom

Goal

  • Write a JS function for each opcode of the NES's CPU.
    (For now, only the implicit, immediate, relative, absolute and absolute+X addressing modes are supported)
    Each function returns its address in the CPU memory (to ease jumps and branches) and outputs asm / binary code.
  • Write JS functions to interact with memory, PPU, APU and controllers (also, later: interact with mappers)
