NES homebrew tools, step 1: display a background tile

Goal

  • Have a 8x8px tile written in the CHR-ROM (16 bytes).
  • Start the program.
  • Wait for 2 frames (the PPU is not ready before that).
  • Write 4 colors into the background subpalette 0.
  • Enable background rendering.
  • The colored tile should be visible on screen (as a mosaic).


Tool #1: Tile editor

This editor allows to edit a single tile.
Click the pixels to loop through colors 0-4.





Tool #2: Palette editor

This editor allows to edit a single background subpalette.
It will be applied to the tile above.
Each color can be picked among the the 64 NES system colors.
